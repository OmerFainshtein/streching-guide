#readMe

how to run the site on your computer:
* make sure you have python installed on your computer
** for now - the site fit only laptop screen

1. download the 'streching-guide' directory from this git page
2. extract the zip folder to a regular folder
2. open cmd on your computer and run: 'cd (the directory path) streching-guide-main'
3. make sure that location on cmd is right (if not, you may have to progress folder by folder using the cd command)
4. in cmd - run 'python -m http.server'
5. open your internet and insert the url 'localhost:8000'
6. enjoy!
